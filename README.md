# Student Spending Analysis

## Objective
Analyze personal spending habits using data visualization.

## Tools Used
- Python
- Pandas
- Matplotlib
- JupyterLab

## Visualizations
- Pie Chart (Expense Distribution)
- Line Chart (Weekly Spending)

## Key Insights
- Highest spending category: Gym.
- Noticeable trend: Increase in the expense at the weekend.
